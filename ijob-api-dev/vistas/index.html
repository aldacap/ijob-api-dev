Todos los servicios se encuentran dentro de la ruta /api
Usuarios
  • get:/autenticar/{usuario}{clave}
  • post:/usuarios/registrar/{nombre}{apellidos}{correo}{cedula}{activo}{direccion}{clave}
Sectores
  • get:/sectores/
  • post:/sectores/{nombre}
Ocupaciones
  • get:/ocupaciones/
  • post:/ocupaciones/{nombre}{descripcion}{_sector}
Imágenes
  • get:/imagenes/
  • get:/imagenes/{idImagen}
  • post:/imagenes/{userPhoto(input-file)}
Ubicaciones
  • get:/Ubicaciones/
  • post:/Ubicaciones/{codigo}{municipio}{departamento}{pais}
Calificaciones (tipo: 1 = otorgadas, 2 = recibidas)
  • get:/calificaciones/{idUsuario}{cantidad}{tipo}
  • post:/calificaciones/{calidad}{respeto}{puntualidad}{observaciones}{idUsuarioOtorga}{idUsuarioRecibe}
Contactos (estado: 0 = solicitud sin contestar, 1 = solicitud aceptada y 2 = solicitud rechazada)
  • get:/contactos/{idUsuario}
  • post:/contactos/{idUsuarioSolicita}{idUsuarioRecibe}
  • put:/contactos/{idContacto}{estado}
