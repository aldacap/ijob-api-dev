Todos los servicios se encuentran dentro de la ruta /api
Usuarios
  • get:/autenticar/{usuario}{clave}
  • post:/usuarios/registrar/{nombre}{apellidos}{correo}{cedula}{activo}{direccion}{clave}
Sectores
  • get:/sectores/
  • post:/sectores/{nombre}
Ocupaciones
  • get:/ocupaciones/
  • post:/ocupaciones/{nombre}{descripcion}{_sector}
Imágenes
  • get:/imagenes/
  • get:/imagenes/{idImagen}
  • post:/imagenes/{userPhoto(input-file)}
